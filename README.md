# How to set up

## Migrations of SQLite3

### add to package.json

```
{
  "type": "module",
}
```

### run command

```
$ node --loader ts-node/esm database/setup.ts
```

### Delete when finished

```
"type": "module",
```

## Launch the project

```
$ npm run dev
```
